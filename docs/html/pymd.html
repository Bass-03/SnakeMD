class pymd.markdown.Header(text: pymd.markdown.InlineText, level: int)¶

Bases: pymd.markdown.Element

A header is a text element which serves as the title for a new section of a document. Headers come in six main sizes which correspond to the six headers sizes in HTML (e.g., <h1>).

Parameters

  • text – the header text

  • level – the header level between 1 and 6

demote()¶

Demotes a header down a level. Fails silently if the header is already at the lowest level (i.e., <h6>).

promote() → None¶

Promotes a header up a level. Fails silently if the header is already at the highest level (i.e., <h1>).

Returns

Nothing

class pymd.markdown.InlineText(text: str, url: Optional[str] = None, bold: bool = False, italics: bool = False, code: bool = False, image: bool = False)¶

Bases: object

The basic unit of text in markdown. All components which contain text are built using this class instead of strings directly. That way, those elements capture all styling information.

Parameters

  • text – the inline text to render

  • url – the link associated with the inline text

  • bold – the bold state of the inline text; set to True to render bold inline text (i.e., True -> bold)

  • italics – the italics state of the inline text; set to True to render inline text in italics (i.e., True -> italics)

  • code – the italics state of the inline text; set to True to render inline text as code (i.e., True -> code)

  • image – the image state of the inline text; set to True to render inline text as an image; must include url parameter to render

render() → str¶

Renders the inline text object as a string. In this case, inline text can represent many different types of data from stylized text to inline code to links and images.

Returns

the InlineText object as a string
